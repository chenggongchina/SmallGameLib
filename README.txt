SGL( Small Game Lib ) is super lightly a game develop lib for silverlight.
SGL provide you a simple lib to develop your own small game quickly on silverlight.

It doesn't do everything for you but provide you a simple framework and some useful tools.
Using SGL you may care more about the logic settings of your game but not game framework.

Enjoy it.
